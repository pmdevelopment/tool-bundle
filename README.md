tool-bundle
===============

Tools for Symfony2

## Install

Add to composer:

```js
  composer require "pmdevelopment/tool-bundle" "dev-master"
```

Add to kernel:

```php
  new PM\Bundle\ToolBundle\PMToolBundle()
```
