tool-bundle
===============

Tools for Symfony2

## Install

Add to composer:

```js
  composer require "pmdevelopment/tool-bundle" "1.3.*"
```

Add to kernel:

```php
  new PM\Bundle\ToolBundle\PMToolBundle()
```
