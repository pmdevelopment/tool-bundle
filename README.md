tool-bundle
===============

Tools for Symfony2

## Install

Add to composer:

```js
  "pmdevelopment/tool-bundle": "1.0.*"
```

Add to kernel:

```php
  new PM\Bundle\ToolBundle\PMToolBundle()
```
